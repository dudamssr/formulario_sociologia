#  Quiz Sociologia

## Descrição do Projeto

Este projeto consiste no desenvolvimento de um aplicativo de quiz utilizando Flutter, baseado no modelo previamente criado nas aulas de MIT App Inventor.

O aplicativo apresenta perguntas ao usuário, permitindo selecionar respostas e acompanhar o resultado ao final do quiz.

---

## Objetivo

Recriar o quiz desenvolvido anteriormente, agora utilizando Flutter, com foco em:

- Prática de desenvolvimento mobile  
- Manipulação de estados  
- Interação com o usuário  
- Estruturação de interfaces  

---

##  Tecnologias Utilizadas

- Flutter  
- Dart  
- Material Design  

---

##  Funcionalidades

- Exibição de perguntas  
- Seleção de alternativas  
- Controle de pontuação  
- Navegação entre perguntas  
- Exibição do resultado final  

---

##  Como Executar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/dudamssr/formulario_sociologia.git
```

2. Acesse a pasta:
```bash
cd quiz-flutter
```

3. Instale as dependências:
```bash
flutter pub get
```

4. Execute o aplicativo:
```bash
flutter run
```

---

##  Funcionamento do Aplicativo

1. O usuário inicia o quiz  
2. As perguntas são exibidas uma por vez  
3. O usuário seleciona uma alternativa  
4. O sistema registra as respostas  
5. Ao final, é exibido o resultado com a pontuação  

---

##  Diferenciais

- Recriação de projeto anterior em nova tecnologia  
- Interface interativa e dinâmica  
- Aplicação de lógica de quiz em Flutter  
- Experiência simples e intuitiva
